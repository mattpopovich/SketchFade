# Privacy Policy version 1.0.0 for the Sketch Fade iOS app

The Sketch Fade iOS app does not collect, store, or share any personal information or user data of any kind. 

The app does not access the Internet at all. It is just a blank canvas.

If you wish to verify this for yourself, see the source code at https://github.com/mattpopovich/SketchFade
